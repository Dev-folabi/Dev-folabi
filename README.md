### 💳 Xedla-pay — Complete Fintech Ecosystem

**The Challenge:** Building a reliable, scalable payment platform that Nigerians can trust for peer-to-peer transactions.

**What I Built:**
* **Microservices Architecture** — Separated concerns into wallet service, transaction service, user service, and notification service
* **Escrow System** — Automated fund holding and release with dispute resolution workflows
* **Balance Management** — Real-time account balancing with transaction history and reconciliation
* **Transaction Engine** — Handles credits, debits, transfers with ACID compliance
* **Webhook Integration** — Payment gateway callbacks for Paystack and Flutterwave

**Tech Stack:** Node.js, Express, MongoDB, Mongoose, Redis, Docker

**What I Learned:** Building fintech taught me that every millisecond matters, every naira must be accounted for, and security isn't just a feature — it's the foundation.

**Status:** 🟢 In Production
